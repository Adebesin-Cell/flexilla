## Quick setup

This guide will help you get started with Flexilla

## Install via npm

<Stepper>
<Step>
### Install the full Libary
<BoxCodeLang text={"Shell"} icon={"shell"}>
```bash
npm i @flexilla/flexilla
```
</BoxCodeLang>
</Step>
<Step>
### Usage
<BoxCodeLang text={"script"} icon={"js"}>
```js
import {Accordion, Tabs} from "@flexilla/flexilla"

const accordion = new Accordion("#my-accordion")

const tabs = new  Tabs("#my-tabs",{ 
    // options
})
```
</BoxCodeLang>
</Step>
</Stepper>

## Using CDN

<Stepper>
<Step>
### Import from CDN
All components
<BoxCodeLang text={"index.html"} icon={"html"}>
```html
<script type="module">
import {
  Tabs,
  Accordion
} from 'https://cdn.jsdelivr.net/npm/@flexilla/flexilla@latest/+esm';

new Tabs("#hego")
new Accordion("#myAccordion")
</script>
```
</BoxCodeLang>

### Import a single Package
<BoxCodeLang text={"index.html"} icon={"html"}>
```html
<script type="module">
import * as flexillatabs from 'https://cdn.jsdelivr.net/npm/@flexilla/tabs@latest/+esm';
new flexillatabs.Tabs("#tabs")
</script>
```
</BoxCodeLang>
</Step>
</Stepper>


## With Module (require)


<BoxCodeLang text={"script"} icon={"js"}>
```js
const Accordion = require('@flexilla/accordion');
new Accordion("#myAccordion")
```
</BoxCodeLang>
