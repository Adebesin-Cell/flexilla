## Quick setup

This guide will help you get started with Flexilla

## Install via npm

<Stepper>
<Step>
### Install the full Libary
<BoxCodeLang text={"Shell"} icon={"/icons/shell.svg"}>
```bash
npm i @flexilla/flexilla
```
</BoxCodeLang>
</Step>
<Step>
### Usage
<BoxCodeLang text={"script"} icon={"/icons/js.svg"}>
```js
import {Accordion, Tabs} from "@flexilla/flexilla"

const accordion = new Accordion("#my-accordion")

const tabs = new  Tabs("#my-tabs",{ 
    // options
})
```
</BoxCodeLang>
</Step>
</Stepper>

## Using CDN

<Stepper>
<Step>
### Include the library via CDN
All components
<BoxCodeLang text={"index.html"} icon={"/icons/html.svg"}>
```html
<head>
    <script src="https://unpkg.com/@flexilla/flexilla/dist/flexilla.min.js"></script>
</head>
```
</BoxCodeLang>

Or a Single Components
<BoxCodeLang text={"index.html"} icon={"/icons/html.svg"}>
```html
<head>
    <script src="https://unpkg.com/@flexilla/accordion/dist/accordion.min.js"></script>
</head>
```
</BoxCodeLang>


</Step>
<Step>
### Initialize the library in your script
Add in body : When Including all components
<BoxCodeLang text={"index.html"} icon={"/icons/html.svg"}>
```html
<body>
    <!-- content here  -->
    <script>
        new Flexilla.Accordion("#my-collapse",{
            // options
        })
    </script>
</body>
```
</BoxCodeLang>

With One Single Component

<BoxCodeLang text={"index.html"} icon={"/icons/html.svg"}>
```html
<body>
    <!-- content here  -->
    <script>
        new Accordion("#my-collapse",{
            // options
        })
    </script>
</body>
```
</BoxCodeLang>
</Step>
</Stepper>


## With Module (require)


<BoxCodeLang text={"script"} icon={"/icons/js.svg"}>
```js
const Accordion = require('@flexilla/accordion');
new Accordion("#myAccordion")
```
</BoxCodeLang>
