## Strucure

### Collapse structure

- `dismissible` : Container of the Element to make dismissible.

- `dissmiss-buttons` : Buttons Element to use as trigger to dismiss Element. Must have `data-dismiss-btn` attribute


The dismissible HTML markup looks like:

```html
<div id="dismiss-me">
    <!-- content here -->
    <button data-dismiss-btn aria-label="remove from screen">
     Dismiss
    </button>
</div>
```

### Consumming the component
Then import the component and init it

```js
import { Dismissible } from '@flexilla/dismissible;
new Dismissible("#dismiss-me")
```

## Remove from dom

The collapse package supports horizontal collapsing. 
Here's how to do 

### Via data-attribute

```js
<div id="dismiss-me" data-action="remove-from-dom" ... >
 
</div>
```

### With param

```js
new Dismissible(
    '#dismiss-me',
    "remove-from-dom"
)
```

## Props/Params & Options

### Params

<BoxTable>
  <BoxTableRow title="dismissible" type="HTMLElement|string">
    Dismissible Container, must be a valid Selector or HTMLElement.
  </BoxTableRow>
  <BoxTableRow title="action" type="remove-from-dom | hide-from-screen | undefined">
    The action to perform when dismissing. Default is `hide-from-screen`
  </BoxTableRow>
  <BoxTableRow title="onDissmiss" type="void">
    Use this method if you want to perform some action on dismissing an Element
  </BoxTableRow>
</BoxTable>

With JS
```js
new Dismissible(
    "#collapse1",
    "hide-from-screen",
    (() => {
        console.log("Dismissed")
    })
)
```


## Methods

### dismiss
You can manually dismiss

```js
import {Dismissible} from "@flexilla/dismissible"

const dismissEl = new Dismissible("#dismiss-me")

dismissEl.dismiss()
```


### Auto Init
This method allow to initialize all dismissible elements. This method receive a string value which help to identify concerned elements(can use class name or data attribute), by default `[data-fx-dismissible]` is used.

```js
import {Dismissible} from "@flexilla/dismissible"

Dismissible.autoInit()  //init dismissible components based on the selector `data-fx-dismissible`
//or
Dismissible.autoInit(".collaspe-item")
```
