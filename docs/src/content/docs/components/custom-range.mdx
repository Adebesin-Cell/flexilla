The **Custom Range Indicator** component allows you to display a dynamic indicator alongside or within an input range element, providing users with immediate visual feedback as they adjust the range value. This component is ideal for enhancing user experience with sliders.

## Installation

To add the **Custom Range Indicator** to your project, install the package using npm:

```bash
npm i @flexilla/range-indicator
```

## HTML Markup

Here's an example of how to structure the HTML for the **Custom Range Indicator**:

```html
<!-- Container Element -->
<div class="range-container" style="position:relative;">
  <input type="range" id="rangeInput" min="0" max="100" value="50">
</div>
```

The **Custom Range Indicator** component listens for changes in the range input value and updates the indicator in real-time to reflect the current value.

### Example Usage

Here's how to initialize and use the **Custom Range Indicator** in a typical setup:

```html
<!-- Range Slider with Indicator -->
<div class="range-container">
  <input type="range" id="rangeInput" min="0" max="100" value="50">
</div>

<script type="module">
  import { CustomRangeIndicator } from '@flexilla/range-indicator';
  
  // Initialize the range indicator
  const rangeIndicator = new CustomRangeIndicator('.range-container', 'range indicator classes');
</script>
```
